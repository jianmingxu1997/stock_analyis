## 概述

### 筛选流程

```
全市场 5000+ 股票
    ↓ (排除 ST/创业板/科创板/北交所/小市值)
1383 只 基础股票池
    ↓ (10 个指标评分)
约 500 只 8 分以上股票
    ↓ (行业 Top5 筛选)
180-220 只 最终选股
```

### 设计理念

1. **分散投资**: 每个行业最多 5 只，避免过度集中
2. **质量优先**: 只选 8 分以上，确保股票质量
3. **趋势向好**: 重点关注均线金叉、MA20 向上等趋势指标
4. **风险控制**: 排除近期大跌的股票（14 日>-7%，20 日>-10%）

## 10 个评分指标

### 指标总览

| 序号 | 指标名称 | 条件 | 分值 | 类型 |
|------|---------|------|------|------|
| 1 | 日线金叉 | MA5 ≥ MA10 | 1 分 | 趋势 |
| 2 | 当日收红 | 涨幅 > 0% | 1 分 | 动量 |
| 3 | RSI<70 | RSI6 < 70 | 1 分 | 超买超卖 |
| 4 | 量比>1 | 量比 > 1.0 | 1 分 | 成交量 |
| 5 | 涨幅<5% | 涨幅 < 5% | 1 分 | 风险控制 |
| 6 | 14 日>-7% | 14 日涨幅 > -7% | 1 分 | 风险控制 |
| 7 | 20 日>-10% | 20 日涨幅 > -10% | 1 分 | 风险控制 |
| 8 | MA20 向上 | 20 日均线向上 | 1 分 | 趋势 |
| 9 | 周线金叉 | 周线 MA5 ≥ MA10 | 1 分 | 趋势 |
| 10 | 3 月>3% | 3 月涨幅 > 3% | 1 分 | 动量 |

**满分**: 10 分  
**及格线**: 8 分  
**优秀**: 9-10 分

### 指标分类

**趋势指标** (40 分): 日线金叉、MA20 向上、周线金叉、RSI<70

**动量指标** (20 分): 当日收红、3 月>3%

**成交量指标** (15 分): 量比>1

**风险控制** (25 分): 涨幅<5%、14 日>-7%、20 日>-10%

## 指标计算公式

### 1. 日线金叉

```javascript
const ma5 = mean(closes.slice(-5));
const ma10 = mean(closes.slice(-10));
const maDiff = (ma5 - ma10) / ma10 * 100;
const pass = maDiff >= 0;  // MA5 ≥ MA10
```

**说明**: MA5 接近或上穿 MA10，短期趋势转好

---

### 2. 当日收红

```javascript
const pass = latest.pct_chg > 0;
```

**说明**: 当日涨幅 > 0%，表示上涨

---

### 3. RSI6 计算

```javascript
let gains = 0, losses = 0;
for (let i = closes.length - 6; i < closes.length; i++) {
    const change = closes[i] - closes[i-1];
    if (change > 0) gains += change;
    else losses -= change;
}
const rsi6 = losses === 0 ? 100 : 100 - (100 / (1 + gains / losses));
const pass = rsi6 < 70;
```

**说明**: RSI<70 表示未进入超买区

---

### 4. 量比计算

```javascript
const avgVol3 = mean(volumes.slice(-3));
const volumeRatio = latestVol / avgVol3;
const pass = volumeRatio > 1.0;
```

**说明**: 量比>1 表示成交量放大

---

### 5. MA20 向上

```javascript
const ma20_today = mean(closes.slice(-20));
const ma20_5daysAgo = mean(closes.slice(-25, -5));
const pass = ma20_today > ma20_5daysAgo;
```

**说明**: 20 日均线向上表示中期趋势向好

## 行业 Top5 策略

### 为什么选择行业 Top5？

**问题**: 如果直接选得分最高的 200 只，可能会集中在某个行业

**解决**: 每个行业最多选 5 只，确保行业分散

### 实现步骤

```javascript
// 1. 按行业分组
const byIndustry = {};
selected.forEach(s => {
    if (!byIndustry[s.industry]) byIndustry[s.industry] = [];
    byIndustry[s.industry].push(s);
});

// 2. 每个行业取 Top5
const finalSelection = [];
Object.values(byIndustry).forEach(industryStocks => {
    industryStocks.sort((a, b) => b.score - a.score);
    const top5 = industryStocks.slice(0, 5);
    finalSelection.push(...top5);
});

// 3. 过滤 8 分以上
const result = finalSelection.filter(s => s.score >= 8);
```
